# PetShop The Best

Site de um petshop com apresentação de produtos, serviços, contato e formulário de cadastro e agendamento.

## Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- GitHub Pages para publicação

## Páginas do site

- `index.html`: página inicial com apresentação do petshop, carrossel de imagens, serviços em destaque e horário de atendimento.
- `produtos.html`: produtos separados nas categorias acessórios, rações não perecíveis e higiene/limpeza.
- `servicos.html`: serviços de banho e tosa, formas de atendimento e tabela de valores.
- `cadastro.html`: formulário de cadastro do cliente, cadastro do pet e agendamento.
- `contato.html`: endereço, telefone, e-mail, horário de atendimento e dúvidas frequentes.

## Funcionalidades

- Menu de navegação responsivo.
- Carrossel de imagens na página inicial.
- Cartões para apresentação de produtos e serviços.
- Formulário com campos de texto, telefone, e-mail, número, checkbox, radio button, select, textarea e datetime-local.
- Campos obrigatórios com `required`.
- Uso de `placeholder`, `pattern`, `min`, `max` e `autofocus`.
- Escolha do serviço: banho ou tosa.
- Escolha do método de atendimento: tele-busca ou entrega do pet no local.
- Agendamento com data e horário.
- Cálculo de valor estimado conforme serviço, porte do pet e tele-busca.
- Validação do horário de atendimento.
- Exibição de mensagem de confirmação no próprio site.
- Imagens em formato JPG armazenadas localmente.
- Recursos de acessibilidade: imagens com `alt`, labels nos campos do formulário, link para pular ao conteúdo principal e atributos ARIA.

## Observação sobre o formulário

O site é estático, adequado para publicação no GitHub Pages. O formulário valida os campos e exibe uma confirmação na tela, mas não grava dados em banco de dados.
